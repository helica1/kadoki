The per-word audio archive imports as one .tar, .tar.gz, or .tar.xz file (the unpacked form is too many individual files to transfer). Tap Import and pick the file:
  • Android: any of .tar / .tar.gz / .tar.xz, picked from the system file browser.

Extraction takes ~30 s for .tar/.tar.gz, longer for .tar.xz (XZ is CPU-heavy); archives we copied in are freed after success.
